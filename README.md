# Smart-Traffic-Management-System
# Features
Pathfinding: BFS, DFS, Dijkstra, A*, Greedy Best-First, Brute-Force
Dynamic Routing: Real-time traffic simulation with weight adjustments
Scheduling: Vehicle assignment (Greedy vs DP/BnB)
TSP Solutions: Nearest Neighbor heuristic & Exact Branch-and-Bound
Visualization: Interactive graph with step-by-step tracing
Analysis: Algorithm comparison with complexity metrics

 # How to Run
pip install streamlit networkx pandas matplotlib numpy
streamlit run app_final.py

# Project Structure
app_final.py
Data Set.xlsx
Upload any Excel file with distance matrix

# Usage
Upload Excel file with distance matrix
Select start and destination locations
Choose algorithm for pathfinding
View interactive graph and step-by-step execution
Compare algorithm performance
Simulate dynamic traffic changes
Optimize vehicle scheduling
Solve TSP for selected locations
